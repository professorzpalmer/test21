Proposed

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issue 76 and Pull Request 141. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

If content can be printed, the following are true:

Resized:

Content can be resized up to 200% without loss of content or functionality;

Changes Reflected:

User changes to letter spacing, line spacing, font family, and color are reflected in the printed content.
