Adapting Text

AA

Proposed

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issues 79, 78, and 74 and in Pull request 124. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

If the technologies being used have the ability to change the default appearance of text (with the exception of images of text and captions, there is no loss of essential content or functionality when the following are changed:

  1. text color and background color to a single different text color and a single different background color
  2. line spacing (leading) to at least 1.5
  3. letter spacing (tracking) to at least 0.12 em
  4. word spacing to at least 0.16 em

Editor's note: The Working Group seeks to include changes in font-family for text as part of the SC, but is not yet able to identify a way to do so that is sufficiently testable.
