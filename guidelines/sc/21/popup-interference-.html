Popup Interference

AA

Proposed

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issue 75. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

For informational content that appears on hover or focus, one of the following is true:

Turn Off
The informational content can be turned off
Not obscure
The informational content does not obsure the triggering content

This does not apply to user agent rendering of the title attribute.
